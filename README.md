
  # Generate Responsive React Components

  This is a code bundle for Generate Responsive React Components. The original project is available at https://www.figma.com/design/iqDbn3XGFfdR6xl6mjv9j7/Generate-Responsive-React-Components.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

Run `npm run dev:server` to start the Express backend API on `http://localhost:4000`.
